## 🚀 Overview

An end-to-end stock price forecasting system combining ARIMA, SARIMA, and LSTM with a Streamlit dashboard.

---

## 🎯 Features

* 📊 Daily & yearly trends
* 🔮 Forecasting using ARIMA, SARIMA & LSTM
* 🧠 Deep learning prediction
* 📥 Download predictions
* ⚡ Interactive dashboard
